## 日历
点击`左右箭头`切换月份，点击`Today`按钮返回现在的月份。
点击`年份`快速选择年份，点击`月份`快速选择月份。
```jsx
import { Calendar } from 'byte-design';
export default () => <Calendar/>
```


## 选择日期
点击`输入框`选择日期。
```jsx
import {InputDatePicker} from 'byte-design/Calendar'
export default () => <InputDatePicker/>
```
